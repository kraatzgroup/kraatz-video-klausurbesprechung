Debug Steps:

  1. Open browser console (F12)
  2. Click "Bewertungen Dashboard" button in navigation
  3. Check console for log messages:
    • "Bewertungen Dashboard Link clicked"
    • "URL tab parameter: ratings"
    • "Setting active tab to: ratings"
  4. Verify URL changes to /admin?tab=ratings
  5. Check if ratings tab becomes active (highlighted)

Expected Behavior:

When clicking "Bewertungen Dashboard":

  • Navigate to /admin?tab=ratings
  • Ratings tab should be highlighted
  • Ratings content should be displayed
  • Console should show debug messages
